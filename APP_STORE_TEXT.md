## 概要（4,000文字以内）

そらもよう——空の写真だけを集めた、新しいSNSアプリ。

毎日見上げる空は、二度と同じ表情を見せません。
朝焼けに染まる空、真夏の青空、夕暮れのグラデーション、雨上がりの虹。
「そらもよう」は、そんなかけがえのない空の一瞬を撮影・編集・共有するためのアプリです。

━━━━━━━━━━━━━━━━━━
■ 主な機能
━━━━━━━━━━━━━━━━━━

【27種類のプロ仕様 編集ツール】
空の写真に最適化された27種類の編集ツールを搭載。
露出・明るさ・コントラストの基本調整から、カーブ調整・HSL調整・二重露光風合成まで、本格的な写真編集がスマホひとつで完結します。

搭載ツール一覧：
露出 / 明るさ / コントラスト / トーン / ブリリアンス / ハイライト / シャドウ / ブラックポイント / 彩度 / 自然な彩度 / 暖かみ / 色合い / シャープネス / ビネット / 色温度 / ホワイトバランス / テクスチャ / クラリティ / かすみの除去 / グレイン / フェード / ノイズリダクション / カーブ調整 / HSL調整 / レンズ補正 / 二重露光風合成 / トリミング・回転

【10種類のフィルター】
ナチュラル / クリア / ドラマ / ソフト / ウォーム / クール / ビンテージ / モノクロ / パステル / ヴィヴィッド
ワンタップで空の雰囲気を自在に変えられます。

【装備システム——お気に入りのツールを厳選】
27種類のツールから5〜8個を「装備」として選択。
自分だけの編集セットをカスタマイズし、よく使うツールにすぐアクセスできます。
表示順もドラッグ&ドロップで自由に並び替え可能。

【空の色・時間帯を自動分析】
投稿した空の写真から、主要な色（最大5色）と撮影時間帯（朝・昼・夕方・夜）を自動で判定。
空の種類（晴天・曇天・夕焼け・朝焼け・嵐）も記録されます。

【色や時間帯で空を検索】
見たい空を色やハッシュタグ、時間帯、空の種類で検索。
「夕焼けの空だけ見たい」「青い空を集めたい」——そんな探し方ができるのは「そらもよう」だけ。

【あなただけの空ギャラリー】
投稿した空の写真はプロフィールにギャラリーとして表示。
日々の空を記録し、あなただけの空コレクションを作りましょう。

【下書き保存】
編集途中の写真は下書きとして保存可能。
いつでも続きから編集を再開できます。

【位置情報の記録】
撮影場所の市区町村レベルの位置情報を記録。
どこで撮った空かが一目でわかります。
